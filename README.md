# pom_frontend

repo is migrated to [bun](https://github.com/oven-sh/bun)

how to run:
`curl -fsSL https://bun.sh/install | bash` for macOS/linux

1. `bun install`

2. `bun run build`

3. `bun run start`

### Contributors

After making changes, you need to  run linter `bun run lint` and format the code with `bun run fmt`
